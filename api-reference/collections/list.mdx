## GET /api/collections

Returns a paginated list of public collections.

### Query Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `page` | 1 | Page number |
| `limit` | 20 | Items per page |
| `search` | — | Search name/description |
| `sort` | `recent` | `recent` or `popular` |
