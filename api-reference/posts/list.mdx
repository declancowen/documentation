## GET /api/posts

Returns a paginated, filterable list of posts.

### Query Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `page` | 1 | Page number |
| `limit` | 20 | Items per page |
| `filmSimulation` | — | Filter by film simulation |
| `cameraModel` | — | Filter by camera model |
| `search` | — | Search title/caption |
| `sort` | `recent` | `recent` or `popular` |
