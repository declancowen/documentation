---
title: Toggle Post Repost
api: POST /api/posts/{id}/repost
description: Repost or unrepost a post. Acts as a toggle.
---

## Authentication

Requires authentication via session cookie.

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | UUID | The post ID |

## Behavior

- If not reposted: creates a `post_reposts` record.
- If already reposted: deletes the record (unrepost).
- The `repost_count` on the post is updated automatically by a database trigger.

## Response

```json
{
  "message": "Post reposted",
  "isReposted": true
}
```

## Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 401 | `UNAUTHORIZED` | Not authenticated |
| 404 | `NOT_FOUND` | Post not found or deleted |
