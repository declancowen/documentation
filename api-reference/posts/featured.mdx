---
title: Featured Posts
api: GET /api/posts/featured
description: Get recently added and most liked posts for the discover page.
---

## Authentication

Optional. If authenticated, includes `isFavorited` status.

## Response

```json
{
  "recentlyAdded": [
    {
      "id": "uuid",
      "caption": "Tokyo street vibes",
      "coverImageUrl": "https://...",
      "cameraModel": "X-T5",
      "filmSimulation": "Classic Chrome",
      "favoriteCount": 24,
      "commentCount": 5,
      "isFavorited": false,
      "createdAt": "2025-01-15T10:30:00Z",
      "author": {
        "id": "uuid",
        "username": "streetshooter",
        "avatarUrl": "https://..."
      }
    }
  ],
  "mostLiked": [...]
}
```

Returns up to 6 posts in each category.
