## GET /api/recipes

Returns a paginated, filterable list of recipes.

### Query Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `page` | 1 | Page number |
| `limit` | 20 | Items per page |
| `filmSimulation` | тАФ | Filter by film simulation |
| `cameraModel` | тАФ | Filter by camera model |
| `search` | тАФ | Search title/description |
| `sort` | `recent` | `recent` or `popular` |
