---
title: Get User Collections
api: GET /api/users/{id}/collections
description: Get collections created by a specific user.
---

## Authentication

Optional. If authenticated and viewing own profile, shows both public and private collections. Otherwise only public.

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | Username or UUID |

## Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | 1 | Page number |
| `limit` | integer | 12 | Items per page |
| `all` | string | — | Set to `true` to fetch all without pagination |

## Visibility Rules

- Own profile: sees all collections (public + private)
- Other users: sees only `is_public = true` collections
