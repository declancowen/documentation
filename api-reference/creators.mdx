---
title: Creators
api: GET /api/creators
description: List creators (users) with sorting by recent, trending, or popular.
---

## Authentication

Optional. If authenticated, includes `isFollowing` status.

## Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | 1 | Page number |
| `limit` | integer | 12 | Items per page |
| `sort` | string | `recent` | Sort order: `recent`, `trending`, or `popular` |

## Sort Options

| Sort | Description |
|------|-------------|
| `recent` | Newest accounts first |
| `popular` | Most total likes (recipes + posts + collections) |
| `trending` | Most engagement in the last 12 hours |

The trending algorithm counts favorites received across all content types within a 12-hour window, with total likes as a tiebreaker.

## Error Responses

| Status | Code | Description |
|--------|------|-------------|
| 400 | `VALIDATION_ERROR` | Invalid sort or pagination params |
