## State Strategy

| State Type | Solution | Location |
|------------|----------|----------|
| Auth | AuthContext | `context/AuthContext.tsx` |
| Auth Overlay | AuthOverlayContext | `context/AuthOverlayContext.tsx` |
| Content Overlay | ContentOverlayContext | `context/ContentOverlayContext.tsx` |
| Dock Layout | DockLayoutContext | `context/DockLayoutContext.tsx` |
| UI Modals | ModalContext | `context/ModalContext.tsx` |
| Save Operations | SaveContext | `context/SaveContext.tsx` |
| Search | SearchContext | `context/SearchContext.tsx` |
| Theme | ThemeContext | `context/ThemeContext.tsx` |
| Notifications | ToastContext | `context/ToastContext.tsx` |
| Server Data | React Query | `hooks/api/*` |
| Form State | React Hook Form | Component-level |
| Local State | useState/useReducer | Component-level |

## Provider Hierarchy

```tsx
<AuthProvider>
  <ModalProvider>
    <ToastProvider>
      <App />
    </ToastProvider>
  </ModalProvider>
</AuthProvider>
```

## Server State with React Query

All API data is fetched and cached via React Query hooks in `hooks/api/`:

```tsx
import { useRecipes } from '@/hooks/api';

const { data, isLoading, error } = useRecipes({
  filmSimulation: 'Classic Chrome',
  sort: 'popular'
});
```

Available hooks:

| Hook | Purpose |
|------|---------|
| `useRecipes` | List/filter recipes |
| `useRecipe` | Single recipe detail |
| `useUser` | User profile data |
| `useFollow` | Follow/unfollow mutations |
| `useFavorite` | Favorite/unfavorite mutations |
| `useCollections` | List/filter collections |
| `useCollection` | Single collection detail |

## Form State

Forms use React Hook Form with Zod validation from the shared package:

```tsx
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { recipeSchema, type RecipeFormValues } from '@recipe-room/shared';

const form = useForm<RecipeFormValues>({
  resolver: zodResolver(recipeSchema),
});
```

## Optimistic Updates

Mutations use optimistic updates for instant UI feedback:

```tsx
const { mutate } = useFavorite();

const handleFavorite = () => {
  mutate(
    { recipeId, isFavorited: !isFavorited },
    { optimisticData: { ...recipe, isFavorited: !isFavorited } }
  );
};
```

If the server request fails, the UI automatically rolls back.
