## Prerequisites

- Node.js 18+
- pnpm 8+
- A Supabase project ([supabase.com](https://supabase.com))
- iOS Simulator (for mobile development)

## Installation

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone <repo-url>
    cd recipe-room-monorepo
    pnpm install
    ```
  </Step>
  <Step title="Set up environment variables">
    ```bash
    cp apps/web/.env.example apps/web/.env.local
    cp apps/mobile/.env.example apps/mobile/.env
    ```

    Fill in your Supabase credentials:

    **Web** (`apps/web/.env.local`):
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
    LOOPS_API_KEY=your_loops_api_key
    ```

    **Mobile** (`apps/mobile/.env`):
    ```env
    EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
    EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    EXPO_PUBLIC_LOOPS_API_KEY=your_loops_api_key
    ```
  </Step>
  <Step title="Set up the database">
    Run the initial migration in your Supabase SQL Editor:

    1. Go to your Supabase dashboard → SQL Editor
    2. Paste the contents of `supabase/migrations/001_initial_schema.sql`
    3. Run the query

    See [Supabase Setup](/database/supabase-setup) for full details.
  </Step>
  <Step title="Start developing">
    ```bash
    # Web app
    pnpm dev:web

    # Mobile app (iOS)
    pnpm dev:mobile
    ```
  </Step>
</Steps>

## Available Scripts

| Command | Description |
|---------|-------------|
| `pnpm dev:web` | Start Next.js dev server |
| `pnpm dev:mobile` | Start Expo dev server |
| `pnpm build:web` | Build web app for production |
| `pnpm lint` | Lint all packages |
| `pnpm type-check` | Type check all packages |
| `pnpm test` | Run tests across all packages |
