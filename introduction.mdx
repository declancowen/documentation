## What is Recipe Room?

Recipe Room is a social platform for sharing Fujifilm camera film simulation recipes. Users create, share, and discover custom film simulation settings for Fujifilm X-series cameras, organize them into collections, and engage with the community through posts, follows, and favorites.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Web Frontend | Next.js 14 (App Router), React 18, TypeScript |
| Mobile | Expo SDK 54, expo-router, NativeWind |
| Backend | Next.js API Routes (serverless on Vercel) |
| Database | PostgreSQL via Supabase |
| Storage | Supabase Storage (avatars, recipe-images, post-images) |
| Auth | Supabase Auth (email/password) |
| Email | Loops.so |
| UI | shadcn/ui + Tailwind CSS |
| State | React Query, React Context, React Hook Form + Zod |
| Testing | Jest, React Testing Library, fast-check |

## Monorepo Structure

```
recipe-room-monorepo/
├── apps/
│   ├── web/          # Next.js 14 web application
│   └── mobile/       # React Native (Expo) iOS app
├── packages/
│   └── shared/       # Shared types, validation, constants
├── docs/             # Documentation
└── supabase/         # Database schema and migrations
```

The monorepo is managed with pnpm workspaces. The `@recipe-room/shared` package provides types, Zod validation schemas, constants, and API utilities shared between web and mobile.
